Proyecto 'cafeteria-samanez-estrada-productos' (Windows) - Interfaz simple para productos.

Características:
- Lista de productos con columnas: Nombre, Descripción, Precio, Stock.
- Botones: Agregar, Editar, Eliminar.
- Interfaz simple, sin muchos detalles (según pedido).

Requisitos:
- JDK 17 instalado
- Maven (opcional)
- Windows (pom incluye JavaFX classifier 'win')

Ejecutar:
- Desde IntelliJ: abrir carpeta del proyecto, asegurarse Project SDK = Java 17, ejecutar pe.edu.cafeteria.ProductosApplication
- Desde terminal: mvn clean javafx:run
